# Chatbot de la Empresa

## Descripción del Proyecto
Este proyecto consiste en un chatbot diseñado para responder preguntas en nombre de la empresa XYZ. Utiliza datos de la empresa para proporcionar respuestas relevantes y útiles a los usuarios.

## Requisitos
Asegúrate de tener instalado Python en tu sistema. Puedes descargar Python desde [python.org](https://www.python.org/downloads/).
